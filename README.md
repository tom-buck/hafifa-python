# חפיפת Python


ברוכים הבאים לחפיפת פייתון

החפיפה בנויה מחומרים לימודיים ותרגילים:

### חומר
החומר שמצורף הוא חומר מהרשת , שאנחנו חשבנו שהוא יכול לעזור, כול אחד לומד בצורה שונה וקולט אחרת.
לכן אתם מוזמנים ללמוד בכול מקור שתרצו : יוטיוב , כתבות , דוקמנטציות וכול דבר שתמצאו


## תרגילים

תרגילים שיש לבצע. את הפתרונות צריך לעלות לגיט המשותף של הצוות

## חומרים נוספים

תשתמשו בדוקומטציה הרשמית של פייתון בשביל העמקה בדברים.
שיש יהיה את המידע הכי מדויק
היא מגיעה ביחד עם ההתקנה של פייתון
אז אפשר להשתמש בה גם כשאין אינטרנט או ברשת סגורה
